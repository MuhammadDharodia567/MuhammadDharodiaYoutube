SOFTWARE SHORTCUTS CHEAT SHEET:
───────────────────────────────────────────────────────────────

DaVinci Resolve:
- Cut: B
- Delete: Backspace
- Ripple Delete: Shift+Backspace
- Split: Ctrl/Cmd + \
- Zoom Timeline: Alt + Scroll
- Snap: N toggle
- Render: Ctrl/Cmd + R

Premiere Pro:
- Cut: C
- Razor: Ctrl/Cmd + K
- Ripple Delete: Shift+Delete
- Zoom: = and -
- Snap: S toggle
- Render Enter: Return

CapCut:
- Split: Ctrl/Cmd + B
- Delete: Delete
- Speed: Ctrl/Cmd + R
- Freeze: Ctrl/Cmd + Alt + F
